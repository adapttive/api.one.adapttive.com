# api.one.adapttive.com
Serverless API for link preview

# Installation

- Install Serverless Framework

    `npm install -g serverless`

- Build Go Lambda functions

    `make`

- Deploy

    `serverless deploy`

# Endpoints

- Ping

    `https://api.one.adapttive.com/ping/`

- Preview

    `https://api.one.adapttive.com/preview/`
